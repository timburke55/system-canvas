# System Canvas

A modern, web-based tool for visual systems thinking — starting with Causal Loop Diagrams and qualitative simulation.

## Getting Started

```bash
npm install
cp .env.example .env
# Fill in your environment variables in .env
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Tech Stack

- **Next.js 15** (App Router)
- **React Flow v12** for the interactive canvas
- **Auth.js v5** with Google sign-in
- **Prisma 7** with PostgreSQL
- **Zustand** for state management
- **Tailwind CSS 4** with shadcn/ui-style components
